# Femto-Text-Editor

A CLI-based text editor written in **C++**

---

## Features

* **Edit / Insert / Delete Lines**
* **Undo Operations**
* **Copy / Paste Buffer Between Files**
* **Various File Statistics**
* **File Similarity Checker**
* **Find, Search, and Replace**
* **File Compression for Large Text Files**
* **Recent Files Opened**
* **Autosave on Unexpected Program Exit**

---
# How to Use the Text Editor

## The program provides the following operations at startup:

* -1: Exit
* 0: Read a file
* 1: Create a file and edit it
* 2: Open a file and edit it
* 3: Copy/Paste text from file
* 4: Search in file
* 5: Display file statistics
* 6: Search and modify
* 7: Print the command palette
* 8: Check file similarity
* 9: Show recent files
* 10: Compress a large file
* 11: De-compress a large file

# For operations 1 and 2, in-editor commands are available:

* /cmd – return to main menu
* /e – edit a specific line
* /i – insert a line at a specific line number
* /d – delete a range of lines
* /D – delete a single line
* /u – undo the last operation

# For operation 3:

* /c – copy text between line numbers
* /p – paste text at a specified line

# For operation 4:

* /f – find a specific word (case-insensitive)
* /m – find the most frequent word

# For operation 6:

* /r – find and replace
* /! – find and delete all occurrences
